Maç Analizi & Geçmiş

Riot ID ile oyuncu sorgulama, arama geçmişi. Maç sayısı, win rate, KDA, ADR, rank, RR özeti. Kaydırmalı liste, renk kodlu galibiyet/mağlubiyet. 10 oyuncu istatistiği, takım karşılaştırma, yarı skorları, highlight'lar. Mod/zaman/harita/sıralama filtreleri.

Oyuncu Sorgulama Maç Geçmişi Filtreleme
📈
Gelişmiş İstatistikler

6 detay paneli: Win rate, ADR trend, KD ratio, HS%, First blood, ACS. RR/ELO değişim çizgi grafiği, rank atlama işaretçileri. Harita başına performans, aktivite ısı haritası, clutch başarısı (1v1-1v5) ve multi-kill (ace/4k/3k/2k) sayıları.

RR/ELO Grafiği Aktivite Takvimi Clutch & Multi-kill
🔌
Riot API & Ayarlar

Riot ID ile hesap sorgulama, maç geçmişi ve detayı (10 oyuncu, round istatistikleri, ability cast), toplu istatistik hesaplama. API key DPAPI ile şifreli saklama. Açık/Koyu tema, Türkçe/İngilizce dil, cihaz seçimi. GitHub Releases ile otomatik güncelleme, Firebase RTDB bildirimleri.
